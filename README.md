# 🔊Text To Speech Converter

## Overview
The **Text To Speech Converter** is a simple web-based application that converts written text into spoken audio using the browser’s built-in speech synthesis feature. Users can type any text, choose a voice, and listen instantly.

## Features

### Text Input
- Accepts any text entered by the user
- Large text area for easy typing

### Voice Selection
- Loads available system voices automatically
- Allows users to choose different voices from the dropdown

### Speech Conversion
- Converts entered text into speech instantly
- Plays the speech when the **Listen** button is clicked

### User Interface
- Clean and modern design
- Responsive layout with gradient background

## Technologies Used
- **HTML5** → Structure of the web page
- **CSS3** → Styling and layout
- **JavaScript** → Speech synthesis logic and interactivity
- **Web Speech API** → Converts text into voice

## Project Structure
- **index.html** → Main structure of the application
- **style.css** → Styling and responsive design
- **script.js** → Handles voice loading and speech conversion

## Software Requirements
- Any modern web browser (**Chrome, Edge, Firefox, etc.**)
- Code editor (**VS Code recommended**)

## How It Works
1. User types text in the text area
2. Available voices are loaded automatically
3. User selects a preferred voice
4. On clicking **Listen**, the browser speaks the entered text

## How to Run the Project
1. Download or clone the project
2. Open the project folder
3. Run the **index.html** file in your browser
